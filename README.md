# Network-Intrusion-Detection-Using-Multi-Agent-Autonomous-AI-system
Overview

This project presents a multi-agent cybersecurity system designed to detect, analyze, and respond to network threats in a structured and explainable manner. The system integrates machine learning, rule-based reasoning, and large language models to simulate the workflow of a real-world Security Operations Center (SOC).

Instead of relying on a single model, the system uses multiple intelligent agents that collaborate to process network traffic, identify threats, and generate actionable insights.

Key Features
Multi-agent architecture for modular and scalable design
Machine learning-based intrusion detection
Rule-based behavioral analysis for explainability
AI-driven reasoning using CrewAI
Aggregated threat analysis for efficient decision-making
Real-time or simulated network monitoring dashboard
Structured outputs for detection, analysis, and response
